호이스팅(hoisting): 함수가 선언이 유효한 범위 내에서 제일 위쪽으로 함수의 선언을 끌어올려준다.

> 함수 선언문: `function hello() {}`<br/>
> 함수 표현식: `const hello = function () {}`
>
> 호이스팅은 함수 선언에서만 발생한다.

```jsx
hello();

function hello() {
  console.log("Hello");
}
```

hello()가 read 될 시점에서 hello라는 이름의 함수가 정의되어 있지 않으나 작동에는 문제가 없다. => **hoisting되었다**
